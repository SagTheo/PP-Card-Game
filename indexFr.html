Règles du jeu

×
  • Lorsque le bouton 'Jouer' est cliqué, une nouvelle partie commence
  • Vous aurez 5 secondes pour mémoriser les nombres sur les cartes avant qu'ils ne disparaissent
  • Vous devrez ensuite cliquer sur une carte pour révéler le nombre
  • Si vous trouvez une paire, les nombres resteront sur les cartes
  • Sinon, tous les nombres disparaîtront
  • Si vous trouvez toutes les paires, vous gagnez. Cela ajoutera 1 au compteur des parties gagnées
  • Si vous faites disparaître les nombres 3 fois, vous perdez. Cela ajoutera 1 au compteur des parties perdues
  • Si vous perdez 3 fois, c'est game over. Les compteurs des parties gagnées et perdues seront remis à zéro
